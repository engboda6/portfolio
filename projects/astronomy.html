Cosmic Observer

Desktop astronomy app for Windows: interactive 3D sky, near-Earth object tracking, orbital debris, dark-matter lensing maps, and an AI astronomy tutor — packaged as a native .exe.

View on GitHub Quick start

Repository

engboda6 / cosmic-observer https://github.com/engboda6/cosmic-observer

Open source (MIT). Clone, run with Node.js 18+, or build the Windows installer and portable exe.

Highlights

  • 3D Sky — live celestial sphere from GPS or manual location
  • Sky Events — sunrise/sunset, moon phases, ISS pass estimates
  • NEO Tracking — NASA NeoWs feed with risk scoring
  • Orbital Debris — stations & debris via CelesTrak
  • Dark Matter — lensing Δκ map + probabilistic black-hole zones
  • AI Tutor — OpenAI chat grounded in app context (mock without a key)

The dark-matter / black-hole section is a research-style probabilistic model (Myr–Gyr scales), not a deterministic future-date prediction.
