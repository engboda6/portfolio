Project 02

Vertical Farm Telemetry Analysis

A pure Python data analysis system for a vertical farm. It reads a real telemetry ledger with 600 production rows and produces a clean dataset, analysis tables, charts, Excel exports, and a delivery-ready PDF report.

Pipeline (one command)

  1. Import & clean — prepare_dataset.py
    Reads vertical_farm_biomass_telemetry_ledger.csv, cleans columns, parses Stack / Rack / Row IDs, outputs an analysis-ready dataset.
  2. Analysis — analysis.py
    Nutrient decay acceleration matrix for 60 irrigation channels, weak-circulation channels, harvest-weight index per crop, best nutrition balance from top yield rows.
  3. Charts — charts.py
    EC distribution, circulation vs nutrient decay, matrix heatmap, harvest index plot.
  4. Report — report.py
    3-page PDF (Abd Alrahman Ahmed) with summary, matrix, and recommendations.
  5. Full run — run_pipeline.py
    Runs every step above with a single command.
