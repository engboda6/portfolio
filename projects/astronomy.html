Cosmic Observer

A Windows desktop astronomy observatory built with JavaScript end-to-end: React + Three.js for a live 3D sky, Node.js/Express for local APIs and orbital calculations, and Electron for a native .exe — powered by real NASA, JPL, and CelesTrak data.

Stack overview

Core language is JavaScript — React + Three.js on the frontend, Node.js/Express on the backend, wrapped as a Windows desktop app with Electron.

Core languages

  • JavaScript (JSX)
  • HTML + CSS
  • PowerShell
  • GLSL

Frontend

  • React 18
  • Three.js
  • Vite

Backend

  • Node.js
  • Express
  • astronomy-engine
  • OpenAI SDK
  • cors / dotenv

Desktop packaging

  • Electron 33
  • electron-builder
  • NSIS installer (.exe)
  • Portable build

Live data sources

  • NASA NeoWs
  • JPL Sentry
  • JPL CAD
  • NASA CNEOS Fireball
  • CelesTrak (TLE)
  • VizieR / CDS

Features

  • Interactive 3D sky (stars, planets, moons, Milky Way)
  • Realistic sky dome shading (GLSL)
  • Celestial positions via astronomy-engine
  • Sunrise / sunset and moon phases
  • Near-Earth objects & impact risk (NASA / JPL)
  • Satellite tracking with TLE (CelesTrak)
  • Star cluster data from VizieR / CDS
  • AI astronomy guide chatbot (OpenAI)
  • Windows location via PowerShell
  • Desktop app with installer and portable builds
