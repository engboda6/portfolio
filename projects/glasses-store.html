Online Store

A full-stack online store: storefront, cart, auth, admin dashboard, inventory, and a camera-based virtual try-on experience.

View on GitHub

Repository

engboda6 / glasses-store https://github.com/engboda6/glasses-store

Next.js App Router project with storefront pages, REST API routes, Prisma schema, and admin tools for products, banners, offers, and site settings.

Project summary

Goal: give an eyewear shop a complete online selling flow — browse products, add to cart, manage stock and content from an admin panel, and let shoppers preview frames on their face before buying.

What it includes

  • Storefront with product listing, product detail pages, and shopping cart
  • Customer registration / login and admin authentication APIs
  • Admin dashboard for products, inventory, orders, banners, offers, and settings
  • Image upload pipeline for product and brand assets
  • Virtual try-on using face detection in the browser
  • Bilingual / theme-aware storefront UI

Tech stack

  • TypeScript
  • Next.js
  • React
  • Prisma
  • Tailwind CSS
  • face-api.js
