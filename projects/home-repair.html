Home Repair Loyalty Analysis

A home-repair loyalty project focused on post-service surveys and tiered service bundles, built from real marketing data in Data.xlsx. Author: Eng: Abd Alrahman Ahmed.

Open PDF

Project summary

Goal: help a home-repair business grow returning customers through surveys after every completed job and tiered service bundles (Standard / Seasonal / Premium). Final deliverable: a dark visual report exported as a fully black 5-page PDF with no white margins.

Key findings

  • 2,000 clean rows · ~$1.56M revenue · 3.02x ROAS · 49.4% returning customers
  • Email has the strongest retention (51% returning)
  • Referral × Premium is the strongest loyalty pair (55.6%)
  • Standard has the highest average ticket but the lowest return rate → survey priority

How it was built

  • Inspected Data.xlsx with Python + openpyxl
  • Loyalty metrics by month, channel, region, and service type
  • Interactive analysis board with KPI cards, tables, and charts
  • Standalone dark HTML report with SVG charts
  • Chrome Headless export → fully black 5-page PDF
  • PyMuPDF page PNGs for portfolio / GitHub preview

Client recommendations

  • Post-service surveys within 24 hours of job completion
  • Priority: Standard jobs and East region
  • Score ≤3 → callback · Score ≥4 → one-click bundle offer
  • Care Plus — two seasonal checkups after a Standard repair
  • Premium Shield — annual plan for Referral / returning Premium
  • Seasonal Autopilot — email reminder + pre-booked Seasonal visit

Tech stack

  • Python
  • openpyxl
  • PyMuPDF
  • HTML / CSS / SVG
  • Chrome Headless
